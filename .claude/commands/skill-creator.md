새로운 스킬(커맨드)을 생성해줘.

아래 순서대로 진행해:

## 1단계: 정보 수집

사용자에게 아래 항목을 질문해:

1. **스킬 이름**: 슬래시 명령어로 사용할 이름 (영문, 소문자, 하이픈 허용. 예: `my-skill`)
2. **설명**: 이 스킬이 하는 일을 한 줄로 설명 (예: "코드 리뷰 체크리스트 생성")
3. **범위**: 이 프로젝트 전용인지, 모든 프로젝트에서 쓸 건지
   - 프로젝트 전용 → `.claude/commands/`
   - 글로벌 → `~/.claude/commands/`
4. **동작 내용**: 스킬이 실행되면 어떤 작업을 해야 하는지 상세히 설명

## 2단계: 스킬 파일 생성

수집한 정보를 바탕으로 `.md` 파일을 생성해. 기존 스킬 패턴을 따를 것:

- 파일 경로: `{범위에 따른 디렉토리}/{스킬이름}.md`
- 프롬프트는 한국어로 작성
- 명확한 지시문과 출력 형식 포함
- 필요 시 단계별 구조로 작성

## 3단계: 등록 확인

파일 생성 후:
1. 파일이 정상 생성되었는지 확인
2. 사용법 안내: `/{스킬이름}` 으로 실행 가능
3. CLAUDE.md의 스킬 목록에 추가가 필요하면 안내

## 규칙
- 기존 스킬과 이름이 중복되지 않도록 확인
- 스킬 내용은 사용자가 설명한 동작을 충실히 반영
- 과도한 구조화 금지 — 간결하고 실용적으로 작성
