### 1-3. 구현 완료 기능
- 5단계 위자드 (주제→키워드→스타일→사진→아웃라인+생성)
- AI 키워드 분석 + 경쟁 블로그 분석
- 16개 카테고리별 맞춤 글 생성 (5가지 톤)
- 실시간 SEO 체크리스트 (12항목)
- AI 부분/전체 재작성, 도입부 최적화
- 사진 SEO 3요소 (파일명, ALT, 캡션) 자동화
- 썸네일 자동 생성 (5스타일, 커스텀 폰트)
- 가독성 분석 + AI 감지 분석 (자연스러움 점수)
- 워너비 스타일 분석 (스크린샷 기반)
- 협찬/체험단 가이드 파싱 + 준수 체크 (일부)
- 소셜 로그인 (네이버/카카오/구글)
- 무료 체험 (월 3회) + 가입 첫 달 무료 프로모션
- 히스토리 대시보드 + 버그 리포트

## 2. 경쟁사 분석

### 2-1. 경쟁 구도 매핑

```
                    네이버 SEO 특화
                         ↑
                         |
          가제트AI ●      | ● 피클잇
                         |
    대량 생성 ←─────────┼──────────→ 품질/편집
                         |
          워들리 ●        |        ● 뤼튼
                         |
                         ↓
                    범용 글쓰기
```

### 2-2. 경쟁사 비교표

| 기준 | 피클잇 | 가제트AI | 워들리 | 뤼튼 | ChatGPT |
|------|--------|---------|--------|------|---------|
| **입력 방식** | 사진 업로드 | 키워드 | 키워드 | 텍스트 | 텍스트 |
| **네이버 SEO** | O (12항목 실시간) | O | 부분 | X | X |
| **사진 분석→본문** | O (유일) | X | X | X | X |
| **내장 에디터** | O (TipTap) | X | X | X | X |
| **경쟁 블로그 분석** | O | X | X | X | X |
| **가독성/자연스러움** | O (이중 분석) | X | X | X | X |
| **체험단 가이드 모드** | O (유일) | X | X | X | X |
| **대량 생성** | X | O | O | X | X |
| **AI 이미지 생성** | X | O | O | O | O |
| **가격 (무제한)** | ₩4,900 (BYOK) | ~₩9,900/월 | ~₩9,000/월 | 유료 | $20/월 |

### 2-3. 피클잇만의 차별점 (3가지 핵심)

**1. 사진 기반 입력 — 유일무이**
- 경쟁사 전부 "키워드 입력"으로 시작
- 피클잇만 "사진 올리면 끝" → 진입 장벽이 가장 낮음
- 사진에서 추출한 정보가 글에 반영 → "진짜 경험 기반" 글

**2. End-to-End 워크플로 — 올인원**
- 사진 분석 → 키워드 → 경쟁 분석 → 아웃라인 → 본문 → SEO 코칭 → 썸네일
- 경쟁사는 "생성 후 복붙" 방식, 피클잇은 에디터 안에서 완결

**3. 체험단/협찬 가이드 모드 — 경쟁사 전무**
- 가이드 업로드 → AI 파싱 → 준수 체크 → 맞춤 글 생성
- 체험단 블로거 30~50만명의 직접적 pain point 해결

### 2-4. 피클잇의 약점
| 약점 | 경쟁사 대비 | 대응 방향 |
|------|-----------|----------|
| 대량 생성 불가 | 가제트AI·워들리는 하루 수십 개 | 품질 포지셔닝 유지 (양 < 질) |
| AI 이미지 생성 없음 | 경쟁사 대부분 보유 | Phase 3 검토 또는 비핵심 판단 |
| 인지도 0 | 가제트AI는 유튜브 마케팅 활발 | GTM 전략 필요 (3단계에서 수립) |
| 결제 미구현 | 경쟁사 모두 결제 가능 | 긴급 구현 필요 |

## 3. 시장 환경 분석

### 3-1. 시장 규모
| 항목 | 규모 |
|------|------|
| 네이버 블로그 월간 활성 사용자 | 약 3,000만+ |
| 체험단/협찬 활성 블로거 | 약 30~50만명 |
| 국내 인플루언서 마케팅 시장 | 약 8,000억~1조원 (2024) |
| 글로벌 AI Writing 시장 | 약 $3B → $10B+ (2028, 연 30%+) |

### 3-2. 핵심 트렌드

**1. AI 글쓰기 도구 폭발적 성장**
- 2024→2025 AI 블로그 도구 사용자 급증
- ChatGPT/Gemini 직접 활용 사용자도 증가 → 전용 도구의 가치 = "편의성 + SEO 특화"

**2. 네이버의 AI 콘텐츠 정책**
- AI 사용 자체는 페널티 아님
- 핵심 기준: "유용한 콘텐츠인가" (C-Rank, D.I.A. 알고리즘)
- 대량 자동 생성 + 유사 패턴 반복 → 저품질 판정 위험
- **피클잇 포지셔닝 유리**: "AI 보조 + 사용자 편집" = 저품질 회피

**3. 체험단 시장 지속 성장**
- 소상공인 마케팅 예산이 블로그 체험단으로 이동 중
- 경기 불황에도 견조 (저비용 마케팅)
- 블로거 1인당 월 2~10건 참여, 가이드 준수가 최대 고통점

**4. 사용자가 가장 원하는 것 (우선순위)**
1. 네이버 상위노출 (SEO)
2. AI 티 안 나는 자연스러운 문체
3. 사진 포함 완성 글
4. 키워드 분석/추천
5. 빠른 생성 속도
6. 네이버 에디터 호환 (복붙 서식 유지)
7. 저품질 회피 안심

## 5. 핵심 인사이트 & 액션 포인트

### 인사이트 1: "사진 기반"은 마케팅 킬 메시지
- 경쟁사 전부 "키워드 입력" → 피클잇만 "사진만 올리면 끝"
- 이게 가장 직관적이고 쉬운 입력 방식
- **마케팅 메시지**: "글쓰기 실력 필요 없어요. 사진만 올리세요."

### 인사이트 2: 체험단 모드는 PMF(Product-Market Fit) 후보
- 30~50만 체험단 블로거, 건당 2~3시간 소요
- 가이드 파싱 + 준수 체크 = 직접적 시간 절약
- **경쟁사에 이 기능 없음** → 독점 니치 가능
- **마케팅 채널도 명확**: 레뷰, 미블 등 체험단 플랫폼 커뮤니티

### 인사이트 3: 결제 구현이 모든 것보다 선행
- 아무리 마케팅해도 돈을 받을 수 없으면 의미 없음
- 무료 사용자만 늘면 → API 비용만 증가
- **권장**: 마케팅 본격화 전에 최소 BYOK 결제 구현

### 인사이트 4: 29명 → 100명은 "채널"이 아니라 "입소문"
- 현 단계에서 광고비 지출은 비효율
- 베타 7명의 실사용 경험 → 후기 → 커뮤니티 공유가 가장 효과적
- **핵심 질문**: "베타 사용자가 주변에 추천할 만큼 만족하는가?"

### 인사이트 5: 초기 타겟은 "체험단 블로거" 1개로 집중
- 3개 페르소나를 동시에 잡으면 메시지가 흐려짐
- 체험단 블로거 = 니즈 명확 + 커뮤니티 존재 + 입소문 잘 퍼짐
- Phase 1 마케팅에서 이 그룹에만 집중 → PMF 검증 후 확장
