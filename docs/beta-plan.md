## 2. 베타 신청 플로우

```
랜딩 → CTA 클릭 → 구글 로그인 (네이버/카카오 "준비 중" 비활성)
  → 베타 코드 입력 화면
     │
     ├─ 코드 입력 + 이름 + 소속 + 개인정보 동의
     │   → 검증 → 사용 안내 팝업 → 서비스 (베타, 7일)
     │
     └─ "먼저 무료로 체험하기"
         → 바로 서비스 (무료 체험, 3회/월)
```

### 베타 코드
- 코드: `PIKLITVIP`
- 인원 상한: 100명
- 100명 초과 시: 마감 안내 + 대기 신청 폼

### 수집 정보 (코드 보유자만)
- 이름 (구글 프로필 자동 채움, 수정 가능)
- 소속 (없으면 "소속없음" 기재)
- 이메일 (구글 로그인 자동)

### 개인정보 동의
- 체크박스 + "자세히 보기" 펼침
- 수집 항목: 이름, 소속, 이메일
- 수집 목적: 클로즈드 베타 서비스 제공 및 안내
- 보유 기간: 정식 서비스 출시 후 1개월 이내 파기

| 기능 | 무료 체험 | 베타 테스터 (7일) | Pro (정식) |
|------|:---------:|:-----------------:|:----------:|
| 글 생성 | 3회/월 | 3회/일 | 무제한 |
| 키워드/경쟁/도입부/CTA 등 | O | O | O |
| 썸네일 생성 | X (추후 논의) | O | O |
| AI 이미지 생성 | X | 5장/7일 | 무제한 |

### 기간
- 코드 입력일부터 7일
- 서비스 사정에 따라 베타가 조기 종료될 수 있음 안내
- 연장: 별도 연장 코드 입력으로 7일 추가

### 만료 알림 (D-2)
- 방식: 앱 내 모달
- 내용: 잔여 기간 안내 + 연장 코드 입력 + 무료 전환 안내

### 종료 후 전환
- 자동으로 무료 체험 (3회/월)
- 정식 런칭 시 할인 쿠폰 제공 (이메일 발송)
