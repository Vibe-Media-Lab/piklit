# 피클잇 (Piklit) — CDO 전략 문서

> Chief Digital Officer 관점의 서비스 비전·전략·로드맵

## 서비스 비전 & 미션

**비전**: 누구나 전문 블로거처럼 글을 쓸 수 있는 세상

**미션**: 사진만 올리면 검색 상위 노출되는 블로그 글이 완성되는 AI 도구를 만든다.

**한 줄 소개**: 사진을 글로 절이다 — 사진 업로드만으로 SEO 최적화된 네이버 블로그 글을 자동 생성하는 AI 서비스

## 타겟 사용자 페르소나

### 페르소나 1: 소상공인 사장님 (40대, 카페 운영)
- **니즈**: 가게 홍보 블로그를 쓰고 싶지만 글쓰기에 자신 없음
- **현재**: 사진만 찍어서 인스타에 올리고, 블로그는 포기
- **기대**: 사진 올리면 "00카페 방문 후기" 같은 완성된 글이 나옴
- **핵심 가치**: 시간 절약, 전문가 수준 글 품질

### 페르소나 2: 초보 블로거 (20대, 대학생)
- **니즈**: 네이버 블로그 수익화를 시작하고 싶음
- **현재**: SEO가 뭔지는 알지만, 키워드 분석이나 구조화가 어려움
- **기대**: 키워드 추천 → 아웃라인 → 본문까지 AI가 가이드
- **핵심 가치**: SEO 학습, 빠른 글 생산

### 페르소나 3: 숙련 블로거 (30대, 체험단 활동)
- **니즈**: 하루 2~3개 포스트를 효율적으로 작성
- **현재**: 직접 쓰지만 반복 작업(키워드, 구조, 도입부)에 시간 소모
- **기대**: AI가 골격을 잡아주고, 본인은 경험과 감성만 더함
- **핵심 가치**: 생산성 극대화, 커스터마이징 자유도

## 핵심 가치 제안 5가지

1. **사진 → 글 자동 변환**: 사진만 업로드하면 AI가 분석하고 완성된 글 생성
2. **네이버 SEO 특화**: 키워드 밀도, 소제목 구조, 도입부 최적화까지 네이버 알고리즘에 맞춤
3. **실시간 SEO 코칭**: 작성 중 체크리스트로 점수 확인, 부족한 부분 즉시 개선
4. **카테고리별 전문 템플릿**: 맛집/쇼핑/여행/육아 등 16개 카테고리별 최적화 프롬프트
5. **AI + 사람의 협업**: AI가 골격을 잡고, 사용자가 경험과 감성을 더하는 하이브리드 워크플로

## 비즈니스 모델: BYOK 하이브리드

### 수익화 3단계

| 단계 | 모델 | 설명 |
|------|------|------|
| 무료 체험 | 서버 API 키 공유 | 신규 사용자에게 글 생성 3회/월 제한, AI 이미지 차단 |
| BYOK | Bring Your Own Key | 사용자가 자신의 Gemini API 키 입력 + 월 구독 |
| Pro | 월 결제 | 프리미엄 기능 (클라우드 저장, 예약 포스팅, 고급 분석) |

### 요금 구조 (확정)
- **Free**: 월 3회 글 생성, AI 이미지 차단, 분석류 무제한
- **BYOK (월 ₩4,900)**: 본인 API 키 + 무제한 생성·이미지
- **Pro (월 ₩18,900)**: 클라우드 저장, 예약 포스팅, 고급 경쟁 분석, 팀 멤버, 우선 지원

### 런칭 프로모션
- **가입 첫 달 무료**: 신규 가입 후 30일간 서버 키로 무제한 사용 (글 생성 + AI 이미지 포함)
- 프로모션 종료 후 → 무료 체험(3회/월)으로 전환, BYOK 또는 Pro 구독 유도

### 핵심 차별 요소
1. **사진 기반 입력**: 유일하게 사진에서 글을 생성. "진짜 경험 기반" 글이라는 네이버 SEO 핵심 가치에 부합
2. **End-to-End 워크플로**: 사진 → 키워드 분석 → 경쟁 분석 → 아웃라인 → 본문 생성 → SEO 코칭까지 한 곳에서
3. **내장 에디터**: TipTap 에디터에서 AI 버블 메뉴 편집. 경쟁사는 모두 생성 후 복붙 방식
4. **체험단/협찬 가이드 모드**: 가이드 업로드 → AI 파싱 → 가이드 준수 글 자동 생성 + 실시간 체크 (경쟁사 전무)
5. **BYOK 최저가**: 경쟁사 최소 월 9,000~9,900원 vs 피클잇 ₩4,900 + 가입 첫 달 무료

## 상세 기획: 협찬/체험단 글 모드

> Phase 2 핵심 기능. 네이버 블로거 수익화의 가장 큰 페인포인트를 해결.

### 배경
- 네이버 체험단·협찬 블로거는 업체로부터 "가이드"를 받음
- 가이드에는 필수 키워드, 필수 문구, 사진 포함 요건, 링크 삽입 등이 명시
- **현재 페인포인트**: 가이드 보면서 글 쓰고, 다 쓴 후 하나하나 체크 → 누락 시 수정 반복 (1건당 2~3시간)

### 핵심 플로우 2가지

#### 플로우 A: 가이드 기반 글 생성 (작성 전)
```
가이드 업로드 → AI 파싱 → 내 사진 업로드 → 가이드 준수 글 자동 생성
```

1. **가이드 업로드**: 이미지(캡처) 또는 텍스트 붙여넣기
2. **AI 파싱**: Gemini가 가이드에서 필수 요소 자동 추출
   - 필수 키워드 (메인/서브)
   - 필수 포함 문구 (업체 슬로건, 이벤트 문구 등)
   - 사진 요건 (최소 장수, 특정 컷 필수 여부)
   - 링크 삽입 요건 (업체 홈페이지, 예약 링크 등)
   - 해시태그 요건
   - 금지 표현 (경쟁사 언급 금지 등)
   - 광고 표시 요건 (문구 위치·형태)
3. **사용자 확인**: 추출된 요소를 카드로 보여주고 수정 가능
4. **사진 업로드**: 기존 위자드 플로우 재사용
5. **가이드 준수 글 생성**: AI가 필수 요소를 모두 반영한 글 생성
   - 프롬프트에 가이드 요소 주입
   - 필수 키워드 자연스럽게 배치
   - 필수 문구 본문에 삽입
   - 광고 표시 자동 삽입 (위치·형태 가이드 준수)

#### 플로우 B: 가이드 준수 체크 (작성 후)
```
이미 쓴 글 + 가이드 업로드 → AI가 준수 여부 체크 → 미충족 항목 수정 제안
```

1. 기존에 작성한 글을 에디터에서 열기
2. 가이드 업로드 (플로우 A와 동일)
3. **실시간 준수 체크리스트** (SEO 체크리스트와 유사한 UI):

```
┌─────────────────────────────────────┐
│  📋 가이드 준수 체크  [8/10 충족]    │
├─────────────────────────────────────┤
│  ✅ 필수 키워드 "○○호텔" — 3회 포함  │
│  ✅ 필수 키워드 "강남 호텔" — 2회     │
│  ✅ 필수 문구 "공식 홈페이지 예약"    │
│  ✅ 사진 10장 이상 — 12장 포함       │
│  ✅ 객실 사진 포함                   │
│  ✅ 조식 사진 포함                   │
│  ✅ 광고 표시 "업체로부터 제공..."    │
│  ✅ 해시태그 5개 이상                │
│  ❌ 예약 링크 미삽입 → [자동 삽입]   │
│  ❌ "가격 대비" 표현 금지 위반       │
│     → 2번째 문단에서 발견 [수정하기]  │
└─────────────────────────────────────┘
```

4. 미충족 항목 클릭 → 해당 위치로 이동 + AI 수정 제안

### 기술 구현 방향

| 요소 | 구현 방식 |
|------|-----------|
| 가이드 파싱 | Gemini에 이미지/텍스트 전달 → JSON 추출 (기존 AIService 패턴) |
| 가이드 저장 | Firestore (재사용: 같은 업체 가이드 다시 쓸 때) |
| 준수 체크 | 클라이언트 사이드 텍스트 매칭 + AI 보조 (금지 표현 등 맥락 판단) |
| 광고 표시 삽입 | 에디터 상단/하단에 자동 블록 삽입 (TipTap extension) |
| 프롬프트 | 기존 fullDraft 프롬프트에 가이드 요소 컨텍스트 추가 |

### 과금 배치
- **무료**: 체험단 모드 월 1회 (맛보기)
- **BYOK/Pro**: 무제한 + 가이드 저장·재사용

## 리스크 평가 & 대응

### 기술 리스크
| 리스크 | 영향 | 대응 |
|--------|------|------|

| 네이버 SEO 알고리즘 변경 | 높음 | 프롬프트 기반이므로 빠른 대응 가능, 정기 모니터링 |
