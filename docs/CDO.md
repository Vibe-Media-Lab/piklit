# 피클잇 (Piklit) — CDO 전략 문서

> Chief Digital Officer 관점의 서비스 비전·전략·로드맵

## 서비스 비전 & 미션

**비전**: 누구나 전문 블로거처럼 글을 쓸 수 있는 세상

**미션**: 사진만 올리면 검색 상위 노출되는 블로그 글이 완성되는 AI 도구를 만든다.

**한 줄 소개**: 사진을 글로 절이다 — 사진 업로드만으로 SEO 최적화된 네이버 블로그 글을 자동 생성하는 AI 서비스

## 타겟 사용자 페르소나

### 페르소나 1: 소상공인 사장님 (40대, 카페 운영)
- **니즈**: 가게 홍보 블로그를 쓰고 싶지만 글쓰기에 자신 없음
- **현재**: 사진만 찍어서 인스타에 올리고, 블로그는 포기
- **기대**: 사진 올리면 "00카페 방문 후기" 같은 완성된 글이 나옴
- **핵심 가치**: 시간 절약, 전문가 수준 글 품질

### 페르소나 2: 초보 블로거 (20대, 대학생)
- **니즈**: 네이버 블로그 수익화를 시작하고 싶음
- **현재**: SEO가 뭔지는 알지만, 키워드 분석이나 구조화가 어려움
- **기대**: 키워드 추천 → 아웃라인 → 본문까지 AI가 가이드
- **핵심 가치**: SEO 학습, 빠른 글 생산

### 페르소나 3: 숙련 블로거 (30대, 체험단 활동)
- **니즈**: 하루 2~3개 포스트를 효율적으로 작성
- **현재**: 직접 쓰지만 반복 작업(키워드, 구조, 도입부)에 시간 소모
- **기대**: AI가 골격을 잡아주고, 본인은 경험과 감성만 더함
- **핵심 가치**: 생산성 극대화, 커스터마이징 자유도

## 핵심 가치 제안 5가지

1. **사진 → 글 자동 변환**: 사진만 업로드하면 AI가 분석하고 완성된 글 생성
2. **네이버 SEO 특화**: 키워드 밀도, 소제목 구조, 도입부 최적화까지 네이버 알고리즘에 맞춤
3. **실시간 SEO 코칭**: 작성 중 체크리스트로 점수 확인, 부족한 부분 즉시 개선
4. **카테고리별 전문 템플릿**: 맛집/쇼핑/여행/육아 등 16개 카테고리별 최적화 프롬프트
5. **AI + 사람의 협업**: AI가 골격을 잡고, 사용자가 경험과 감성을 더하는 하이브리드 워크플로

## 비즈니스 모델: BYOK 하이브리드

### 수익화 3단계

| 단계 | 모델 | 설명 |
|------|------|------|
| 무료 체험 | 서버 API 키 공유 | 신규 사용자에게 글 생성 3회/월 제한, AI 이미지 차단 |
| BYOK | Bring Your Own Key | 사용자가 자신의 Gemini API 키 입력 + 월 구독 |
| Pro | 월 결제 | 프리미엄 기능 (클라우드 저장, 예약 포스팅, 고급 분석) |

### 요금 구조 (확정)
- **Free**: 월 3회 글 생성, AI 이미지 차단, 분석류 무제한
- **BYOK (월 ₩4,900)**: 본인 API 키 + 무제한 생성·이미지
- **Pro (월 ₩18,900)**: 클라우드 저장, 예약 포스팅, 고급 경쟁 분석, 팀 멤버, 우선 지원

### 런칭 프로모션
- **가입 첫 달 무료**: 신규 가입 후 30일간 서버 키로 무제한 사용 (글 생성 + AI 이미지 포함)
- 프로모션 종료 후 → 무료 체험(3회/월)으로 전환, BYOK 또는 Pro 구독 유도

### 핵심 차별 요소
1. **사진 기반 입력**: 유일하게 사진에서 글을 생성. "진짜 경험 기반" 글이라는 네이버 SEO 핵심 가치에 부합
2. **End-to-End 워크플로**: 사진 → 키워드 분석 → 경쟁 분석 → 아웃라인 → 본문 생성 → SEO 코칭까지 한 곳에서
3. **내장 에디터**: TipTap 에디터에서 AI 버블 메뉴 편집. 경쟁사는 모두 생성 후 복붙 방식
4. **BYOK 최저가**: 경쟁사 최소 월 9,000~9,900원 vs 피클잇 ₩4,900 + 가입 첫 달 무료

## 3단계 로드맵

### Phase 1: MVP (현재 ~ 배포)
**목표**: 핵심 기능 완성 + 첫 배포

- [ ] 랜딩 페이지 (piklit.pro)
- [ ] 소셜 로그인 (네이버/카카오/구글)
- [ ] API 키 서버 사이드 숨김 (Firebase Functions)
- [ ] 무료 체험 (사용량 제한)
- [ ] Firebase Hosting 배포
- [x] 위자드 기반 글 작성 (4단계)
- [x] AI 키워드 분석 + 경쟁 분석
- [x] AI 본문 생성 (16개 카테고리)
- [x] 실시간 SEO 분석 (12개 항목)
- [x] AI 부분 재작성 (4모드)
- [x] 도입부 최적화 (3전략)
- [x] 가독성 분석
- [x] 이미지 ALT 텍스트 생성
- [x] 히스토리 대시보드

### Phase 2: 성장 (배포 후 1~3개월)
**목표**: 사진 차별점 극대화 + 리텐션 강화

#### 사진 차별점 강화 (클라이언트 구현)
- [ ] 사진 SEO 최적화 — 파일명, alt 텍스트, 이미지 설명 자동 생성 (Gemini)
- [ ] 사진 → 썸네일 자동 생성 — Canvas API 크롭/리사이즈 + 제목 텍스트 오버레이
- [ ] 사진 묶음 → 여러 글 분리 — 다수 사진 업로드 시 AI가 주제별 그룹핑 + 각각 글 생성
- [ ] 다음 글 추천 — 작성 히스토리 기반 AI 주제/키워드 제안

#### 기존 계획
- [ ] 글 품질 개선 (더 자연스러운 본문, 톤 매칭 정밀화)
- [x] 시즌/트렌드 키워드 추천 기능
- [ ] 내부 링크/시리즈 관리
- [ ] 클라우드 저장 (Firestore)
- [ ] 사용자 피드백 기반 프롬프트 튜닝

### Phase 3: 수익화 + 소상공인 확장 (3~6개월)
**목표**: BYOK + 구독 모델 안착 + 발행 후 단계 + 소상공인 특화

#### 수익화
- [ ] BYOK 모드 (사용자 API 키 입력)
- [ ] 구독 결제 시스템 (월간/연간)
- [ ] 프리미엄 기능 분리
- [ ] 팀/기업 요금제

#### 리텐션 — 발행 후 단계 (Firebase Functions 필요)
- [ ] 발행 후 성과 추적 — 검색 순위, 조회수 변화 대시보드 + 알림
- [ ] 경쟁 매장 블로그 모니터링 — 같은 업종 주변 경쟁 블로그 추적 + 차별화 키워드 제안

#### 소상공인 특화 (Firebase Functions 필요)

## 리스크 평가 & 대응

### 기술 리스크
| 리스크 | 영향 | 대응 |
|--------|------|------|
| Gemini API 가격 변동 | 높음 | BYOK 모델로 비용 전가, 멀티 모델 지원 준비 |
