# 피클잇 수동 테스트 체크리스트

> 테스트 URL: https://piklit.vercel.app/
> 작성일: 2026-03-12
> 테스트 브라우저: Chrome (PC) + Safari (iPhone)

---

## 1. 베타 테스터 시스템

### 베타 코드 입력
- [ ] 로그인 → 마이 메뉴 → 설정 → 베타 코드 입력란 보이는지
- [ ] `PIKLIT-VIP` 입력 → "베타 활성화" 성공 토스트
- [ ] 활성화 후 D-day 배지 표시되는지 (예: "베타 D-6")
- [ ] 잘못된 코드 입력 → 에러 메시지

### 베타 플랜 적용
- [ ] 베타 활성화 후 글 생성 → 무제한 작동 (3회 제한 없음)
- [ ] AI 이미지 생성 → pro 전용이므로 베타에서도 차단되는지 확인

## 2. 버그 리포트

### 버그 리포트 버튼
- [ ] 로그인 후 좌측 하단 🐛 FAB 버튼 보이는지
- [ ] (모바일) FAB이 탭바 위에 위치하는지
- [ ] 비로그인 시 FAB 숨겨지는지

### 버그 리포트 제출
- [ ] 🐛 클릭 → 모달 열림
- [ ] 설명 입력 → "전송" 클릭
- [ ] "버그 리포트가 전송되었습니다" 토스트
- [ ] 빈 설명으로 전송 시도 → "문제 설명을 입력해주세요" 경고

### 관리자 확인
- [ ] /admin/bugs 접속 → 방금 제출한 리포트 보이는지
- [ ] 상태 변경 (신규→확인→해결) 동작

### Safari 호환성
- [ ] iPhone Safari에서 🐛 클릭 → 모달 열림
- [ ] 스크린샷 자동 첨부 동작 (html2canvas Safari 호환)
- [ ] 전송 성공

## 3. 전체 플로우 E2E

### 로그인
- [ ] Google 로그인 성공
- [ ] (선택) 카카오 로그인 성공
- [ ] (선택) 네이버 로그인 성공
- [ ] 로그인 후 글 목록 페이지 이동

### 위자드 Step 1: 주제 선택
- [ ] "새 글 쓰기" → 위자드 시작
- [ ] 카테고리 선택 (예: 맛집)
- [ ] 주제 입력 → "다음"

### 위자드 Step 2: 키워드
- [ ] AI 키워드 분석 버튼 → 결과 표시
- [ ] 키워드 선택/해제 동작
- [ ] 직접 키워드 입력 → 추가
- [ ] (선택) 시즌 키워드 분석

### 위자드 Step 3: 스타일 설정
- [ ] 경쟁 분석 버튼 → 결과 표시
- [ ] 톤앤무드 선택 → 샘플 미리보기
- [ ] 글자수/문단호흡 설정
- [ ] (선택) 워너비 스타일 분석

### 위자드 Step 4: 이미지
- [ ] 사진 업로드 (2~3장)
- [ ] 사진 분석 → 결과 표시
- [ ] 프로그레스 바 업데이트
- [ ] "사진 없이 진행" 버튼 동작

### 위자드 Step 5: 아웃라인 + 생성
- [ ] 아웃라인 생성 → 편집 가능
- [ ] "본문 생성" → 스트리밍 표시
- [ ] 생성 완료 → 에디터 전환
- [ ] 축하 카드 표시

### 에디터
- [ ] 본문 표시 정상 (이미지 포함)
- [ ] 제목 입력/수정
- [ ] 툴바 동작 (Bold, Italic, H2, H3)
- [ ] SEO 체크리스트 표시
- [ ] 복사 버튼 → 클립보드 복사 성공

### 이탈 방지
- [ ] 에디터에서 뒤로가기 → 경고 표시

## 4. AI 편집 도구 4종 (2026-03-11 구현)

### 4-1. 인라인 TIP 카드
- [ ] 우측 사이드바 → 인체감 분석 → "분석" 클릭
- [ ] 제안 항목 클릭 → 본문 해당 위치 형광펜 하이라이트
- [ ] TIP 카드 표시 (수정안 + "적용"/"닫기")
- [ ] "적용" → 본문 텍스트 교체
- [ ] "닫기" → TIP 사라짐

### 4-2. SEO 자동수정
- [ ] SEO 체크리스트에서 미통과 항목 확인
- [ ] "AI SEO 자동 수정 (N건)" 버튼 보이는지
- [ ] 클릭 → 자동 수정 실행 → 결과 반영
- [ ] AI 배지 표시

### 4-3. 워너비/내스타일
- [ ] 위자드 Step 3에서 워너비 스타일 분석
- [ ] 스크린샷 슬롯 업로드 → 분석 실행
- [ ] 프리셋 저장 → 다음 글에서 불러오기
- [ ] "내스타일" 탭 전환 동작

### 4-4. AI 전체 재작성
- [ ] 에디터 상단 "AI 전체 재작성" 버튼 보이는지
- [ ] 클릭 → confirm 대화상자
- [ ] 확인 → 재작성 실행 → 새 본문 반영

---

## 5. SEO 점수 비교 카드 (2026-03-11 구현)

- [ ] AI 전체 재작성 후 → 점수 비교 카드 표시
- [ ] 이전 점수(취소선) + 이후 점수(볼드) + 변화량 배지
- [ ] 상승 시 초록 배지, 하락 시 빨강 배지
- [ ] "되돌리기" 버튼 → 원본 복원
- [ ] 카드 닫기 → confirm 경고 ("원본 복원 불가")

## 6. 에디터 상단 설정 변경 바 (2026-03-11 구현)

- [ ] 에디터 상단에 현재 설정 표시 (카테고리, 톤, 글자수 등)
